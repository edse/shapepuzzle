This is a breakthrough the new HTML5 technologies using a simple puzzle game.

In this game we are using a few new HTML5 technologies:

  • Canvas
  • Image manipulation
  • RequestAnimationFrame
  • Audio API
  • Video API
  • Application Cache API
  • Mouse, keyboard, and touch support
  • Fullscreen API
